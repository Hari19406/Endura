# Endura

Endura is an AI-powered adaptive running coach that generates personalized workouts based on fatigue,readiness and performance data.

## Features
-Dynamic workout generation
-Fatigue and readiness scoring
-Personalized pacing 
-Goal based training (5K, 10K, HM, FM)

## Tech(planned)
-Mobile: React Native/ Flutter
-Backend: Node.js/ FastAPI

## Vision
Build a smarter alternative to Runna with real adaptive intellignece.
