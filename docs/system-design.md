# Endura System Design

## Components

1. Mobile App
- Collects user input
- Displays workouts

2. Backend
- Stores user data
- Sends workout plans

3. AI Engine
- Generates workouts
- Calculates fatigue & readiness

## Flow

User → Mobile App → Backend → AI Engine  
AI Engine → Backend → Mobile App → User

## Goal
Create a fully adaptive running coach.
